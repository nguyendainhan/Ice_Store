# 🧊 Ice Store Web

Website bán đá lạnh được xây dựng bằng **Vue 3 + Vite**.

## 📌 Giới thiệu

Ice Store Web là website giới thiệu và bán sản phẩm đá lạnh trực tuyến.  
Website bao gồm các trang:

- Trang chủ (Home)
- Giới thiệu cửa hàng
- Thành viên
- Quy định cửa hàng
- Trang mua sắm
- Đăng nhập / Đăng ký
- Giỏ hàng

---

## 🚀 Công nghệ sử dụng

- Vue 3
- Vite
- Vue Router
- JavaScript
- HTML5 & CSS3

---
